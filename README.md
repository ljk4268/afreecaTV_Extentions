## 📺 프로젝트 소개
'오늘의 공유'는 방송을 시작하면, 공유할 일정 및 정보를 시청자가 빠르게 확인할 수 있도록 효율적으로 구성해주는 확장프로그램입니다. <br />
예를 들어, 방송 컨텐츠 1부, 2부, 3부, 일정 등 / 합방 멤버 정보 / 공지 / 립밤 이름은 뭐에요? 마우스, 키보드 어떤 거 쓰세요?와 같은 중복 질문을 방지하는 기능을 확장 프로그램에서 확인할 수 있습니다.
